# WDD430 - Next.js Learning

## Project Overview

This repository contains coursework for the [Next.js Dashboard App tutorial](https://nextjs.org/learn/dashboard-app) from nextjs.org. The project builds a full-stack financial dashboard application to learn core Next.js features and modern web development practices.

## What We're Building

A financial dashboard application with:
- Public home page
- Login page with authentication
- Protected dashboard pages
- Invoice management (add, edit, delete)
- PostgreSQL database integration

## Learning Objectives (First 5 Chapters)

### Chapter 1: Getting Started
- Create Next.js application using starter template
- Explore project structure (`/app`, `/public`, config files)
- Understand TypeScript setup and type definitions
- Run local development server

### Chapter 2: CSS Styling
- Different styling approaches in Next.js
- Working with CSS modules, Tailwind CSS, and global styles

### Chapter 3: Optimizing Fonts and Images
- Next.js Image component for automatic optimization
- Font optimization with `next/font`

### Chapter 4: Creating Layouts and Pages
- File-system based routing
- Nested layouts and pages
- Route organization

### Chapter 5: Navigating Between Pages
- Client-side navigation with `<Link>` component
- Active link states and navigation patterns

## Tech Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS (pre-styled components provided)
- **Database**: PostgreSQL (setup in later chapters)
- **Authentication**: NextAuth.js (later chapters)
- **Package Manager**: pnpm
